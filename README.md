# UI library

Component Library for the project spacio. We can use this repo as a master tree of all components required in the project. Tokens, component styles, motion etc can be specified here independently and copied into the master project for testing in context of the main product.

## Development server

Run `ng serve` for a dev server. Navigate to `http://localhost:4200/`. The application will automatically reload if you change any of the source files.

## Further help

To get more help on the Angular CLI use `ng help` or go check out the [Angular CLI Overview and Command Reference](https://angular.io/cli) page.
